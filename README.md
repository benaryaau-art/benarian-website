# BENARIAN Premium Website

A responsive luxury travel website ready for GitHub Pages and Netlify.

## Upload to GitHub
1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, and `script.js` to the repository root.
3. Commit the files.
4. For GitHub Pages: Settings → Pages → Deploy from branch → main / root.
5. For Netlify: drag this entire folder or ZIP into Netlify Drop.

## Notes
- High-resolution images are loaded from Unsplash CDN.
- Replace demo booking actions with Booking.com affiliate/API links when credentials are available.
- Visa guidance is a design preview and should be updated from official embassy sources before publication.
